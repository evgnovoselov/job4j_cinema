Проект Cinema!

Вас приветствует проект сервиса Кинотеатра, данный проект предоставляет возможность ознакомиться с фильмами, которые проказываются на экране и есть возможность купить билеты. Для покупки билета нужно зарегистрироваться или авторизоваться в системе, перейти в раздел расписания и выбрать необходимый сеанс фильма.

Доступные разделы:

  • Расписание
  • Кинотека
